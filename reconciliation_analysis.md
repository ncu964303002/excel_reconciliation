# Excel 勾稽比對 - 結果分類與結論建議

## 一、常見的勾稽結果分類

### 1. **完全匹配 (Perfect Match)**
- 所有檔案都存在該 Key
- 所有欄位值完全一致
- 狀態：✅ 正常

### 2. **Key 存在但值不一致 (Value Mismatch)**
- 所有檔案都有該 Key
- 但某些欄位的值不同
- 狀態：⚠️ 需要確認
- 常見原因：
  - 資料更新時間不同
  - 來源系統計算邏輯差異
  - 人工輸入錯誤

### 3. **部分檔案缺失 (Partial Missing)**
- 只在部分檔案中存在
- 狀態：⚠️ 資料不完整
- 常見原因：
  - 某系統尚未同步
  - 資料範圍不一致
  - 篩選條件差異

### 4. **單一來源 (Single Source)**
- 只在一個檔案中存在
- 狀態：❌ 嚴重差異
- 常見原因：
  - 獨有的新增資料
  - 其他系統已刪除
  - 資料遺漏

### 5. **金額差異 (Amount Difference)**
- Key 存在，但數值型欄位有差異
- 需要計算差異金額或百分比
- 狀態：⚠️ 需調整
- 通常需要：
  - 顯示差異金額
  - 計算差異百分比
  - 標註超過容差的項目

### 6. **日期不一致 (Date Mismatch)**
- 交易日期、更新日期等時間欄位不同
- 可能影響會計期間
- 狀態：⚠️ 需確認期間

## 二、建議的結論顯示方式

### A. 整體摘要儀表板
```
┌─────────────────────────────────────┐
│     勾稽比對結論報告                │
├─────────────────────────────────────┤
│ 總體勾稽率：85% ✅                  │
│ 勾稽狀態：可接受                    │
├─────────────────────────────────────┤
│ 📊 統計摘要：                       │
│   ✅ 完全匹配：850 筆 (85%)         │
│   ⚠️  值不一致：80 筆 (8%)          │
│   ⚠️  部分缺失：50 筆 (5%)          │
│   ❌ 單一來源：20 筆 (2%)           │
├─────────────────────────────────────┤
│ 💰 金額勾稽：                       │
│   檔案 1 總額：$1,000,000           │
│   檔案 2 總額：$995,000             │
│   差異金額：$5,000                  │
│   差異率：0.5%                      │
├─────────────────────────────────────┤
│ ⚡ 需立即處理：                     │
│   • 20 筆僅存在於單一系統           │
│   • 15 筆金額差異超過 5%            │
│   • 8 筆關鍵欄位為空值              │
└─────────────────────────────────────┘
```

### B. 風險等級分類
- 🟢 低風險：完全匹配，無需處理
- 🟡 中風險：值略有差異，建議確認
- 🟠 高風險：重大差異，需立即處理
- 🔴 嚴重風險：資料遺漏，緊急處理

### C. 可執行的建議
```
📋 建議處理順序：
1. 優先處理 20 筆單一來源資料（可能遺漏）
2. 確認 15 筆超過容差的金額差異
3. 補充 8 筆空值資料
4. 批次審查 80 筆值不一致項目
```

## 三、視覺化建議

### 1. 圓餅圖 - 勾稽狀態分布
### 2. 長條圖 - 各檔案資料完整度
### 3. 差異熱圖 - 哪些欄位最常不一致
### 4. 趨勢線 - 如果有歷史資料，顯示勾稽率趨勢

## 四、匯出報告建議

### Excel 輸出應包含多個工作表：
1. 【摘要】- 整體結論和統計
2. 【完全匹配】- 無問題資料
3. 【值差異】- 需確認的資料
4. 【部分缺失】- 不完整資料
5. 【異常清單】- 需處理的問題
6. 【金額差異明細】- 數值型差異分析
